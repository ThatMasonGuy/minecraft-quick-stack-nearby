Quick Stack Nearby 0.3.2.

- Fixes quick stacking into double chests so matching items in either half can
  fill the full chest.
- Adds a `B`-by-default quick-stack keybind that repeats while held in-world
  and stays inactive while any UI screen is open.
- Fixes right-clicking the quick-stack inventory button so the slot rules
  screen opens correctly across supported Minecraft versions.
- Fixes the remaining Minecraft 26.1.2 rules-screen opener failure caused by
  screen-opening API drift inside the 26.x compatibility group.
