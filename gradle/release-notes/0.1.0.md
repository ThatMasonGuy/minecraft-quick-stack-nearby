Initial 1.21.11 implementation for Quick Stack Nearby.

- Sets up the Fabric build, candidate compatibility profiles, CI workflows,
  smoke-test hooks, and Modrinth publishing guardrails.
- Adds a survival-inventory button that sends a server-authoritative quick-stack
  request.
- Moves matching main-inventory items into nearby containers that already hold
  those item types.
- Uses the Inv+ slot API when available and a local InventorySort-style slot
  fallback otherwise.
