# Gallery Source

Source-controlled Modrinth gallery assets for Quick Stack Nearby.

- Root images under `gallery/QuickStackNearby/` are upload candidates.
- `banner/` contains the selector image for the featured Modrinth banner.
- `description_images/` contains selector copies referenced by
  `gradle/modrinth-project-pages.md`.
- `gallery/metadata.json` supplies titles, descriptions, and ordering.

The original full-size source image remains at repo root as
`quick-stack-nearby.jpg`. The packaged mod icon is a downscaled copy at
`src/main/resources/assets/quick-stack-nearby/icon.jpg`.
